## 新增需求

### 需求：聊天输入
界面必须提供一个文本输入区域供用户输入消息。

#### 场景：发送消息
- **当**用户输入消息并按 Enter 或点击发送时
- **那么**消息被添加到聊天记录中
- **并且**消息被发送到后端
- **并且**输入字段被清空

### 需求：聊天记录
界面必须显示用户和 Agent 的消息滚动列表。

#### 场景：显示消息
- **当**添加新消息（来自用户或 Agent）时
- **那么**它出现在列表底部
- **并且**视图自动滚动到最新消息

### 需求：Markdown 渲染
界面必须使用 Markdown 渲染 Agent 响应。

#### 场景：渲染代码块
- **当** Agent 响应包含代码块 (```) 时
- **那么**它被渲染为语法高亮的代码块
- **并且**用户可以复制代码
