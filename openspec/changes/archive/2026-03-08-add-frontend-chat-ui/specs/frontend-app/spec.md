## 新增需求

### 需求：前端应用程序结构
系统必须提供一个通过 Vite 服务的基于 React 的单页应用 (SPA)。

#### 场景：应用程序初始化
- **当**开发者运行 `npm run dev` 时
- **那么**应用程序在 `localhost:5173` 启动
- **并且**根组件渲染无错误

### 需求：样式系统
应用程序必须使用 Tailwind CSS 进行样式设计。

#### 场景：Tailwind 配置
- **当**组件使用 Tailwind 实用程序类（例如 `bg-blue-500`）时
- **那么**样式在浏览器中正确应用

### 需求：路由
应用程序必须支持客户端路由。

#### 场景：默认路由
- **当**用户访问根 URL `/` 时
- **那么**渲染聊天界面组件
