## 新增需求

### 需求：聊天布局
界面必须使用 TDesign 布局组件（Layout, Content, Header）来构建聊天页面。

#### 场景：响应式布局
- **当**用户调整窗口大小时
- **那么**聊天布局会适当调整（例如，消息列表占据可用高度）

### 需求：消息列表
界面必须使用 TDesign List 或 Card 组件显示滚动的消息列表。

#### 场景：显示消息
- **当**添加新消息时
- **那么**它出现在列表底部
- **并且**视图自动滚动到最新消息
- **并且**用户消息靠右对齐
- **并且** Agent 消息靠左对齐

### 需求：消息输入
界面必须使用 TDesign Input 或 Textarea 组件提供文本输入区域。

#### 场景：发送消息
- **当**用户输入消息并按 Enter（不按 Shift）或点击发送时
- **那么**消息被添加到聊天记录中
- **并且**输入字段被清空

### 需求：Markdown 渲染
界面必须使用兼容 Vue 的 Markdown 渲染器（例如 `markdown-it`）渲染 Agent 响应。

#### 场景：渲染代码块
- **当** Agent 响应包含代码块时
- **那么**它被渲染为语法高亮的代码块
